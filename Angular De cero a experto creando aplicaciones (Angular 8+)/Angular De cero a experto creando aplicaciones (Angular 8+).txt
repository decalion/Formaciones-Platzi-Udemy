Para compilar desde el terminar de visual estudio un archivo de typescript

Utilizamos el comando : tsc nombrearchivo  
por ejemplo "tsc app"

Tambien podemos hacer que cada vez que hacemos un cambio se autocompile sin tener que estar compilando todo el rato. con el modificador -w
por ejemplo "tsc app -w"

con el comando "tsc --init " inicializamos el directorio como un proyecto de TypeScript


Resumen de comandos
tsc nombrearchivo
tsc nombre archivo -w
tsc --init
